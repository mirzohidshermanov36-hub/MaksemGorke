# TaskFlow

Mentor & Support jamoasi uchun landing sayt + CRM (React + Tailwind CSS).

## Ishga tushirish

```bash
npm install
npm run dev
```

Build:
```bash
npm run build
```

## Demo akkaunt

**Manager (CRM):**
- Email: `admin@taskflow.uz`
- Parol: `admin123`

## Qanday ishlaydi

1. Yangi mentor saytda **"Ariza topshirish"** orqali ro'yxatdan o'tadi (email + parol).
2. Ariza menejer panelida **"Arizalar"** bo'limiga tushadi.
3. Menejer arizani **tasdiqlaydi** — shundan so'ng mentor login qila oladi.
4. Menejer mentorga vazifa biriktiradi (nom, deadline, prioritet).
5. Mentor o'z panelida vazifani ko'radi va statusini yangilaydi (Yangi → Bajarilmoqda → Bajarilgan).

Ma'lumotlar `localStorage` da saqlanadi (backendsiz demo).
